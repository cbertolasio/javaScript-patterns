About this Sample

This sample demonstrates how to use the revealing module pattern in javascript to help you provide structure to your code as it grows.

The revealing module pattern is very similar to that of the module pattern except that in the revealing module pattern, EVERYTHING is private until the very end.

  • Namespace
  • Immediate Functions
  • Private and Privileged Functions
  • Declaring Dependencies
